# orcaproxy

A forward web proxy server for security control.

## Description

A forward web proxy server aiming to provide features for security controls.

## Roadmap

* Access control based on:
    * Client IP address ranges
    * Reqeuest method
    * Request header
    * Request body
* HTTPS interception
* HTTP header rewrite
